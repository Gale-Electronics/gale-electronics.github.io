# Community Photos

<p class="lede">
  Real-world 401s and GT2101s from owners and restorers worldwide. Each set is stored at full resolution with the contributor’s name,
  location (optional), and date taken. To submit, email <a href="mailto:vintagegaleuk@gmail.com">vintagegaleuk@gmail.com</a>.
</p>

<div class="note">
  <strong>What to include:</strong>
  <ul>
    <li>Your name (how you’d like to be credited)</li>
    <li>Approximate date/year</li>
    <li>Model (e.g., GS401A, 401C, GT2101)</li>
    <li>Any restoration notes (re-foam, re-cap, stands, finish, etc.)</li>
  </ul>
</div>
